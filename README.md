Sistema de CRUD usando ReactJS, NodeJS e MySql

## :books: Funcionalidades

- <b>Funcionalidade 1</b>: Cadastro de usuário
- <b>Funcionalidade 2</b>: Login do usuário
- <b>Funcionalidade 3</b>: Deslogar da conta
